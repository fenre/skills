# Database Visibility Reference

Primary source:
https://help.splunk.com/en/appdynamics-on-premises/extend-appdynamics/26.4.0/extend-splunk-appdynamics/splunk-appdynamics-apis/database-visibility-api

The renderer emits redacted collector payloads, Database Agent command plans,
and collector/server/node/event validation probes. Do not put a literal
database password in specs. Use `password_file`.
