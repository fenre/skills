# Splunk ITSI Config References

Detailed reference material is split by workflow:

- `references/beginner_quickstart.md` - first service-tree and content-pack examples.
- `references/compatibility.md` - ITSI version and API compatibility notes.
- `references/content_packs.md` - content pack preview and apply behavior.
- `references/native_itsi.md` - native ITSI entities, services, KPIs, and dependencies.
- `references/topology.md` - topology import, service-tree rendering, and glass table helpers.

Start with `SKILL.md` for workflow selection, then open the reference file that matches the requested operation.
